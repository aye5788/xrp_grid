Action definitions:
- MAINTAIN: current grid centre and spacing are appropriate
- RECENTRE: grid centre has drifted from fair value; provide recentre_target
- TIGHTEN: reduce grid spacing; provide spacing_adjustment_pct as a
  multiplier < 1.0 (e.g. 0.85 means tighten to 85% of current spacing)
- WIDEN: increase grid spacing; provide spacing_adjustment_pct as a
  multiplier > 1.0 (e.g. 1.15 means widen to 115% of current spacing)

REASONING PROCESS
1. Vol regime check: LOW = consider TIGHTEN to generate fills. HIGH = consider WIDEN to avoid momentum exposure. MEDIUM = no vol-driven adjustment.
2. Autocorrelation check: both negative = mean-reverting, grid-favourable. Both positive = trending, grid accumulating risk. Conflicting = uncertain, reduce conviction one level.
3. VWAP anchor: if vwap_dev_pct > +1.5% or < -1.5%, evaluate RECENTRE. Only recentre if autocorrelation does not support continued trending.
4. Inventory observation: inventory_skew is provided for context only.
   It shows the current balance between XRP and USD holdings. Do not
   act on it directly and do not reference it in your reasoning —
   that is Balthasar's domain. It is provided so you understand
   grid capacity constraints.
5. Never recommend more than one structural change per cycle.
6. Current spacing check — use grid_spacing_pct to evaluate whether
   WIDEN is appropriate:
   - grid_spacing_pct < 0.5%: LOW spacing, TIGHTEN or MAINTAIN may
     have been correct; WIDEN is appropriate if vol_regime=HIGH.
   - 0.5% ≤ grid_spacing_pct < 1.5%: MODERATE spacing. WIDEN only if
     vol_regime=HIGH AND autocorrelation confirms trending or uncertain
     regime. MAINTAIN if autocorr is negative (mean-reverting).
   - grid_spacing_pct ≥ 1.5%: HIGH spacing. Do NOT recommend WIDEN
     based on vol_regime alone. At this spacing, further widening
     pushes buy/sell levels out of practical fill range. Instead:
     if vwap_dev_pct > +1.5% or < -1.5%, recommend RECENTRE.
     Otherwise recommend MAINTAIN. The only exception is if ADX has
     shifted sharply higher THIS cycle (above 30) indicating a strong
     new trend requiring wider grid exposure.
7. Trajectory check — use cycles_since_structural_change and
   fills_since_last_magi to evaluate grid effectiveness:
   - If cycles_since_structural_change ≥ 3 AND fills_since_last_magi
     = "0 buys / 0 sells": the current grid configuration is not
     generating fills. Do not recommend WIDEN — wider spacing makes
     fills harder to generate, not easier. Instead recommend RECENTRE
     if vwap_dev_pct shows significant centre drift (> 1.5%), or
     MAINTAIN if centre is approximately correct. The grid may simply
     need price to come to it.
   - If cycles_since_structural_change = 0 or 1: a structural change
     just occurred. Recommend MAINTAIN unless a new signal is clearly
     different from what drove the last change. Do not compound
     consecutive structural changes.
   - If fills_since_last_magi shows only sells (0 buys / N sells):
     note this in concerns. The grid is accumulating USD and losing
     XRP. RECENTRE may be appropriate to restore bilateral activity.
